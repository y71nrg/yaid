# ヤフオクIDみえーる

* 2023/10/25のID非表示化改悪対策 chrome://extensions/  > デベロッパー モードをON > パッケージ化されていない拡張機能を読み込む
* 正確には固定された乱数(auc_user_id)を取得している 入札中のユーザの評価チェックなど
* 出品中 出品終了分 ブラックリスト 入札履歴にユーザのユニークIDを表示し、閲覧済みのIDに色を付ける
